# Traveloop – Personalized Travel Planning App

Traveloop is a travel planning web application developed for hackathon purposes using HTML, CSS, JavaScript, and Bootstrap.

## Features

- User Login Page
- Create New Trip
- Add Multiple Cities
- Plan Itinerary
- Budget Estimation
- Activity Planning
- Responsive UI Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Pages

### Home Page
Displays welcome section and navigation.

### Login Page
Allows users to login or sign up.

### Trip Planner
Users can create and manage travel plans.

### Budget Page
Displays estimated travel expenses.

### Itinerary Page
Shows complete travel schedule.

## Project Goal

The goal of Traveloop is to simplify personalized travel planning through an easy-to-use and visually attractive web application.
